# Concordance

KJV verse lists for 64 keywords. Built for **concordance-style study**: pick a key concept, read every passage containing it, distill the verses that condense the divine pattern.

[← home](/) · [→ practices](../practices/)

## How to use

1. Pick a keyword from the lists below
2. Open the corresponding file — every KJV verse containing that word, in canonical order
3. Read every verse
4. Identify the 10-20 that most condense the divine pattern
5. Memorize, chant, and pray them (see [`../practices/`](../practices/))

## Methodology

Verses extracted from a clean KJV plaintext source via regex match on the full English-word-family for each keyword. Each file is sorted in canonical book order (Genesis → Revelation). The regex used appears in the per-file header.

Source: KJV 1769 standard text (public domain).

## Notes

- **Selfish (0 verses)** — the word "selfish" does not appear in KJV. The empty file is preserved deliberately as evidence of the absence.
- **Truth** appears in both lists ([gann-truth](./gann-truth/) + [law-truth](./law-truth/)) — identical content, kept in both for navigational symmetry.
- **JEHOVAH/LORD (6,705 verses)** — every appearance of the Tetragrammaton in KJV. The largest file.
