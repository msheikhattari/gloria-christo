# Practices

How to actually work with scripture. The files in this folder are operational, not theoretical — each prescribes a concrete daily discipline.

## The practices integrated

These are not separate disciplines. They are facets of one practice:

- **Memorization** makes the text portable
- **Chanting** vocalizes the memorized text
- **Praying scripture** routes the memorized + chanted text into intercession
- **Concordance reading** (next door at `../concordance/`) feeds the memorization with the right material

A full daily rhythm using all of them:

1. **Morning** — read the day's concordance key word; identify 1-2 verses to memorize; chant the day's Psalm; pray the verses for those you love
2. **Noon** — brief JEHOVAH chant; mid-day Psalm; pray for one specific person using a scripture promise
3. **Evening** — review the memorized text; chant for sleep; pray the Lord's Prayer slowly
